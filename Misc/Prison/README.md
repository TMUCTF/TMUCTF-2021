```

```
This is Tap (Knock) Code. Just look for online decoders and decode it! Here is the decoding result:  
```
THEPRISONERISTRYINGTOESCAPEFROMHISCELL
```
The flag:  
```
TMUCTF{THEPRISONERISTRYINGTOESCAPEFROMHISCELL}
```
